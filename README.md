# ♊ Gemini Parallel & Search Tool
**让 Gemini 具备“平行推演”与“同屏搜索”能力的交互增强工具。**

目标是让 AI 交互系统同时具备：
- **分层**：主干对话归主干，推演分支归侧边栏，互不干扰。
- **可销毁**：看完即焚，物理级抹除云端历史记录。
- **零摩擦**：光标自动对焦，开启即输入，无需鼠标二次点击。

---

## 核心痛点

在深度使用 Gemini 时，通常会陷入两种坏形态：
1. **主线污染**：为了搞清一个支线概念，在主对话里追问了 20 轮，导致核心任务的上下文被冲淡，AI 记忆偏移。
2. **切窗断流**：为了去 Google 验证一个事实，频繁在浏览器 Tab 之间切换，推演思路瞬间中断。

这个工具的核心设计是按 **“交互维度”** 拆分功能：
- **💡 Parallel Branch**：捕获当前上下文，在侧边栏开启独立推演沙盒。
- **🔍 Contextual Search**：在同屏侧边栏内直接驱动 Google 搜索。
- **🎯 Auto-Focus**：利用 DOM 节点注入技术，实现“指令 + 换行 + 对焦”的瞬间衔接。

---

## 快速开始

### 1. 开发者模式安装
1. 下载并解压本仓库。
2. 在 Chrome 地址栏访问 `chrome://extensions/`。
3. 开启右上角 **“开发者模式”**。
4. 点击 **“加载已解压的扩展程序”**，选中文件夹。

### 2. 验证路径
按以下顺序验证核心链路：
1. **划选**：确认划词菜单弹出。
2. **注入**：点击 `💡`，确认光标自动在引导语下方的空行闪烁。
3. **销毁**：对话后点击 `遗忘分支`，观察侧边栏自动化点击过程。
4. **清理**：刷新主页面，验证左侧历史记录中无该分支残留。
